# TypeScript Basics

A learning project for practicing the fundamentals of TypeScript.

## Technologies

- TypeScript
- Vite
- Axios
- Prettier

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
